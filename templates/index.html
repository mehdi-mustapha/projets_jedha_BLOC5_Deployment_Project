API Documentation : prédiciton de la qualité du vin

Endpoints

Ce point final vous permet de faire une prédiction de la qualité d'un vin, de 1 à 10, en fonction de certains paramètres

.

Comment demander ce point de terminaison

Vous devez demander ce point de terminaison en utilisant une méthode POST avec un contenu JSON. Votre JSON doit contenir les valeurs, séparées par des virgules, des paramètres suivants

  • Paramètre 1 : acidité fixe
  • Paramètre 2 : acidité volatile
  • Paramètre 3 : acide citrique
  • Parameter 4 : sucre résiduel
  • Paramètre 5 : chlorures
  • Paramètre 6 : dioxyde de soufre libre
  • Paramètre 7 : dioxyde de soufre total
  • Paramètre 8 : densité
  • Paramètre 9 : pH
  • Paramètre 10: sulfates
  • Paramètre 11: alcool
Exemple utilisant curl en localhost:
curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' http://localhost:5000/predict

Si vous utilisez l'application déployée dans Heroku: curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://my-appvin-flask-app.herokuapp.com/
